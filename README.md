# 🎓 Online Courses Platform

> Современный адаптивный интерфейс для каталога онлайн-курсов.

![Demo](https://online-courses-tau-coral.vercel.app/)

## 🚀 О проекте

Это проект веб-приложения для просмотра и фильтрации списка онлайн-курсов. Реализована адаптивная верстка (Mobile First), живая фильтрация по категориям и мгновенный поиск без перезагрузки страницы.

## 🛠 Стек технологий

*   **HTML5** (Семантическая верстка)
*   **SCSS** (Препроцессор, переменные, миксины)
*   **JavaScript** (ES6+, Vanilla JS)
*   **Git & GitHub** (Контроль версий)
*   **Vercel** (Деплой)

## ✨ Особенности

*   📱 **Полный адаптив:** Корректное отображение на всех устройствах (от 320px до 4K).
*   🔍 **Живой поиск:** Мгновенная фильтрация карточек при вводе текста.
*   🏷 **Фильтры:** Переключение категорий с анимацией.
*    **БЭМ:** Строгое соблюдение методологии именования классов.

## 📦 Установка и запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ArthurDentPhillip/online-courses.git
    ```

2.  Перейдите в папку проекта:
    ```bash
    cd папка_проекта
    ```

3.  Откройте файл `index.html` в браузере.
    *(Для компиляции SCSS в CSS убедитесь, что у вас установлен SASS)*:
    ```bash
    sass --watch style.scss:style.css
    ```
